## 🚀 快速开始

### 1. 环境准备
```bash
# 安装依赖
pip install -r requirements.txt

# 或快速测试
./run_quick_test.sh
```

### 2. 准备数据
将以下文件放在项目目录:
- `train.csv` - 训练数据
- `test.csv` - 测试数据

### 3. 运行训练
```bash
# 方案1: 神经网络(推荐)
python3 dataset3_solution.py

# 方案2: 逻辑回归(快速)
python3 dataset3_logistic.py
```

### 4. 获取结果
运行完成后会生成:
- `submission.csv` - 可提交至Kaggle的预测结果

## 🛠️ 超参数配置

### 神经网络默认配置
```python
HIDDEN_LAYERS = [128, 64, 32]  # 网络结构
LEARNING_RATE = 0.01           # 学习率
EPOCHS = 200                   # 训练轮数
BATCH_SIZE = 64                # 批次大小
```

### 逻辑回归默认配置
```python
LEARNING_RATE = 0.1            # 学习率
L2_LAMBDA = 0.01              # L2正则化系数
EPOCHS = 1000                  # 训练轮数
BATCH_SIZE = 128               # 批次大小
```

## 📝 使用注意事项

1. **数据格式要求**:
   - CSV格式
   - 训练集包含特征列和标签列
   - 测试集包含特征列和ID列

2. **超参数调整**:
   - 学习率过大会导致不收敛
   - 网络层数过深可能过拟合
   - 建议先用默认参数测试

3. **性能优化**:
   - 可调整网络结构
   - 可增加正则化
   - 可尝试不同优化器

## 🔍 问题排查

### 常见问题

**Q: ModuleNotFoundError: No module named 'numpy'**
```bash
A: pip install -r requirements.txt
```

**Q: 训练时出现nan或inf**
```bash
A: 学习率过大,尝试减小LEARNING_RATE
```

**Q: 准确率不理想**
```bash
A: 1. 增加训练轮数
   2. 调整网络结构
   3. 检查数据预处理
   4. 尝试另一种算法
```

**Q: 数据列名不匹配**
```bash
A: 代码会自动检测常见列名,如'id', 'label', 'target'等
```
