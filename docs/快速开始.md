# Dataset3 快速开始指南

## 🚀 三步完成任务

### 步骤1: 安装依赖 (30秒)
```bash
pip install -r requirements.txt
```

### 步骤2: 准备数据
将以下文件放在项目目录下:
- `train.csv` - 训练数据
- `test.csv` - 测试数据

### 步骤3: 运行程序
```bash
# 推荐: 神经网络方案
python3 dataset3_solution.py

# 或者: 逻辑回归方案(更快)
python3 dataset3_logistic.py
```

**完成!** 查看生成的 `submission.csv` 文件即可提交。

## ⚙️ 超参数调整 (可选)

### 神经网络
打开 `dataset3_solution.py`,修改:
```python
HIDDEN_LAYERS = [128, 64, 32]  # 网络层数和大小
LEARNING_RATE = 0.01           # 学习率(0.001-0.1)
EPOCHS = 200                   # 训练轮数(100-500)
BATCH_SIZE = 64                # 批次大小(32-256)
```

### 逻辑回归
打开 `dataset3_logistic.py`,修改:
```python
LEARNING_RATE = 0.1            # 学习率(0.01-1.0)
EPOCHS = 1000                  # 训练轮数(500-2000)
```

## 🔍 常见问题

**Q: 找不到numpy?**
```bash
pip install numpy pandas
```

**Q: 训练太慢?**
- 减少EPOCHS
- 或使用逻辑回归方案

**Q: 准确率低?**
- 增加EPOCHS
- 调大网络结构(如[256, 128, 64])
- 调整学习率

**Q: 出现nan?**
- 学习率太大,改小(如0.001)
